## Module

| # | Modul | Lernziel | Dauer |
|---|-------|----------|-------|
| 🟢 1 | [GitHub ist nicht was du denkst](https://janrummel.github.io/github-fuer-nicht-programmierer/modul1.html) | Repository erstellen, README schreiben, Commits verstehen | ~20 min |
| 🟢 2 | [Schluss mit v3_final_FINAL](https://janrummel.github.io/github-fuer-nicht-programmierer/modul2.html) | Dateien versionieren, Diffs lesen, alte Versionen einsehen | ~25 min |
| 🟡 3 | [Projekte sichtbar machen](https://janrummel.github.io/github-fuer-nicht-programmierer/modul3.html) | Issues, Labels, Kanban-Board mit GitHub Projects | ~25 min |
| 🟡 4 | [Feedback ohne E-Mail-Ping-Pong](https://janrummel.github.io/github-fuer-nicht-programmierer/modul4.html) | Branches, Pull Requests, Line-by-Line-Reviews | ~30 min |
| 🟠 5 | [In 10 Minuten live](https://janrummel.github.io/github-fuer-nicht-programmierer/modul5.html) | GitHub Pages aktivieren, Website veröffentlichen | ~20 min |
| 🟠 6 | [Dein GitHub-Gameplan](https://janrummel.github.io/github-fuer-nicht-programmierer/modul6.html) | Entscheidungsmatrix, Team überzeugen, nächste Schritte | ~30 min |

## Für wen?

Wissensarbeiter ohne technischen Hintergrund: Projektmanager, Gründer, Wissenschaftler, Ingenieure. Jeder der besser zusammenarbeiten will — ohne Programmieren zu müssen.
